todict
======

.. py:method:: rlemasklib.RLEMask.todict()

   Convert the RLEMask object to an RLE dictionary.

   :returns:

             The RLE dictionary.
                 - ``"size"`` -- [height, width] of the mask
                 - ``"counts"`` -- LEB128-like compressed run-length counts as in pycocotools
