__ior__
=======

.. py:method:: rlemasklib.RLEMask.__ior__(other)

   Compute the union with another RLEMask in place.

   :param other: the other RLEMask

   :returns: Self

   .. rubric:: Examples

   >>> rle1 = RLEMask(np.eye(3))
   >>> rle2 = RLEMask(np.eye(3, k=-1))
   >>> np.array(rle1 | rle2)
   array([[1, 1, 1],
          [1, 1, 1],
          [1, 1, 1]], dtype=uint8)
