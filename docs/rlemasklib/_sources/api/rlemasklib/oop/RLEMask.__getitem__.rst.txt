__getitem__
===========

.. py:method:: rlemasklib.oop.RLEMask.__getitem__(key)

   Crop the RLE mask to get a submask, by slicing.

   :param key: a tuple of two slices, one for height and one for width

   :returns: A new RLEMask object representing the submask.

   .. rubric:: Examples

   With slices:

   >>> rle = RLEMask(np.eye(4))
   >>> rle[1:3, 2:4].shape
   (2, 2)

   With integers:

   >>> rle = RLEMask(np.eye(4))
   >>> rle[1, 1]
   1
