from_dict
=========

.. py:method:: rlemasklib.oop.RLEMask.from_dict(d)
   :staticmethod:


   Create an RLEMask object from an RLE dictionary.

   :param d: RLE dictionary with ``"size"`` and one of ``"counts"``, ``"zcounts"`` and
             ``"ucounts"`` as

             - ``"size"`` -- [height, width] of the mask
             - ``"counts"`` -- LEB128-like compressed run-length counts as in pycocotools
             - ``"zcounts"`` -- zlib-compressed ``counts``
             - ``"ucounts"`` -- uncompressed run-length counts

   :returns: An RLEMask object representing the input mask.
