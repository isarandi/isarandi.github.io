__isub__
========

.. py:method:: rlemasklib.RLEMask.__isub__(other)

   Compute the difference with another RLEMask in place.

   :param other: the other RLEMask

   :returns: Self

   .. rubric:: Examples

   >>> rle1 = RLEMask(np.eye(3))
   >>> rle2 = RLEMask(np.eye(3)[::-1])
   >>> np.array(rle1 - rle2)
   array([[1, 0, 0],
          [0, 0, 0],
          [0, 0, 1]], dtype=uint8)
