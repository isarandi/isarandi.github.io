Index#

class barecat.Index(path, shard_size_limit=None, bufsize=None, readonly=True)#

Bases: contextlib.AbstractContextManager

Manages the SQLite database storing metadata about the files and directories in the Barecat archive.

Parameters:

  • path (str) – Path to the SQLite database file, including the "-sqlite-index" suffix.

  • shard_size_limit (Optional[int]) – Maximum size of a shard in bytes. If None, the shard size is unlimited.

  • bufsize (Optional[int]) – Buffer size for fetching rows.

  • readonly (bool) – Whether to open the index in read-only mode.
