reverse_lookup#

barecat.Index.reverse_lookup(shard, offset)#

Look up a file by its shard and offset.

Parameters:

  • shard (int) – Shard number.

  • offset (int) – Offset within the shard.

Returns:

The file info object.

Raises:

FileNotFoundBarecatError – If the file is not found.

Return type:

BarecatFileInfo
