iter_direct_fileinfos
=====================

.. py:method:: barecat.Index.iter_direct_fileinfos(diritem, order = Order.ANY, bufsize = None)

   Iterate over the file info objects in a directory (non-recursively).

   :param diritem: Directory info object or path of the directory.
   :param order: Order in which to iterate over the files.
   :param bufsize: Buffer size for fetching rows.

   :returns: An iterator over the file info objects.
